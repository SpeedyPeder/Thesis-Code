This GitHub repository contains all numerical examples used in the master's thesis, along with several additional test cases. The repository is organized into three main folders:

\begin{itemize}
    \item \texttt{Additional\_tests}: This folder contains additional numerical tests that are not directly used to produce figures in the thesis, but are included to verify that the implemented schemes behave correctly and produce physically meaningful results.
    
    \item \texttt{CU\_PCCU\_and\_hyperbolicity\_tests}: This folder contains the code used to generate the examples presented in the thesis related to the two-layer shallow-water system solved using both the standard central-upwind flux and the path-conservative central-upwind flux. It also includes the code used for the hyperbolicity correction experiments and the internal dam-break simulation.
    
    \item \texttt{Optimization\_tests}: This folder contains the code used to generate all optimization results presented in the thesis, including optimization of the interface in several one-dimensional test cases, as well as optimization in a two-dimensional rotating system intended to represent a more Earth-like environment.
\end{itemize}
